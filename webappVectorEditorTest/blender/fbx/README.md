# FBX assets (local)

Put `.fbx` files here for Blender workflows—manual import, linking, or future automation from `import_vector_json.py`.

Asset subfolders are named per asset (e.g. `sofa_Beaumont/`). **Unity** `.mat` / `.meta` sidecars are not needed for Blender and are intentionally omitted from this repo.

Large files: consider [Git LFS](https://git-lfs.com/) (add `*.fbx` to `.gitattributes` if needed).
